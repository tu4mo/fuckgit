# f\*ckgit

A fullscreen terminal UI for Git.

<img src="https://github.com/tu4mo/fuckgit/raw/main/screenshot.png" />

## Usage

```sh
npm install -g fckgit
fckgit
```

## Keybindings

| Key            | Action                              |
| -------------- | ----------------------------------- |
| `↑` / `↓`      | Navigate files                      |
| `Space`        | Stage / unstage file                |
| `Tab`          | Switch between files and diff panes |
| `q` / `Ctrl+C` | Quit                                |
